# Legend-Tasker ⚔️
Legend-Tasker is a gamified To-Do List application for Android that transforms daily productivity into an epic quest. Complete your real-life missions, earn gold coins, and customize your hero!

## ✨ Features
- **Quest Board:** Manage your daily tasks with difficulty levels (Easy, Medium, Hard).
- **Gamified Rewards:** Earn 10g, 25g, or 50g per task based on difficulty.
- **Hero Wardrobe:** Spend your hard-earned gold to unlock and equip Common, Rare, Epic, and Legend skins.
- **Reminders:** Built-in alarm system to ensure you never miss a deadline.
- **Custom Environments:** Change your dashboard wallpaper (Forest, Ocean, Sunset, etc.).
- **Quest History:** Keep track of all your conquered missions.

## 🛠️ Built With
- **Language:** Java / Android SDK
- **UI:** Material Design 3 (Glassmorphism theme)
- **Data:** SharedPreferences (via DataManager) for persistent gold and task storage.
